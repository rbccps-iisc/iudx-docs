Register Entity¶

  • Summary: The register entity API is used by the device owners to register new devices with the middleware. Every device must have its own schema, or in other words, meta information about the nature of the information that the device publishes. For more information on schemas visit here

Request¶

  • Endpoint: https://localhost/owner/register-entity

  • Method: POST

  • Security Scheme: Owner credentials

  • Required Headers:

    Header Name Description
    entity Name of the entity being registered

  • Body: A valid catalogue item as mentioned here

  • Example request:
